# Monol Plugin Suite

Claude Code 플러그인 생태계 통합 패키지

## 빠른 설치

```bash
npm install -g monol
```

자세한 설치 방법은 [INSTALL.md](./INSTALL.md)를 참고하세요.

## 포함된 플러그인

| 플러그인 | 설명 |
|---------|------|
| **monol-logs** | 세션 자동저장, 로드맵, 요약, 인덱스 |
| **monol-plugin-scout** | 플러그인 마켓플레이스 스캔/추천 |
| **monol-rulebook** | 규칙 관리 및 동기화 |
| **monol-channels** | 비동기 메시지 교환 |
| **monol-datastore** | 엔터티 기반 데이터스토어 |
| **monol-workbase** | 프로젝트 관리 |
| **monol-x** | AI 퍼스널리티 진화 |
| **monol-design** | CSS 디자인 시스템 |

## 개별 설치

필요한 플러그인만 설치하려면:

```bash
npm install monol-logs
npm install monol-plugin-scout
npm install monol-rulebook
# ... 등
```
